#Prerequisites
* NPM installed
* NodeJs installed

#Install
* clone repo
* cd into workingdir
```sh
npm install
npm start
```

## Change Google API_KEY
* get you an GOOGLE API_KEY for Youtube API V3
* open index.js and change the current key to yours

### Your app will run on localhost:8080
